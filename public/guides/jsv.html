JSplacement: Velvet - Generator settings

Iterations: This controls the value of the boxes drawn onto the canvas multiplied by 2 if using X or Y symmetry. 4 if X&Y. By default this is set to 100.

Sub-iterations: This controls the value of additional boxes drawn onto the canvas in X shape. Like normal iterations, these are multiplied based on symmetry. Default: 50.

Splitter iterations: This controls the amount of boxes drawn onto the canvas in + shape. This value is multiplied by 2 regardless of the symmetry. Default: 50.

Background brightness: Control background color of the map for different results in whatever you're going to do with it.
In d isplacement, setting this to 0 will make everything extrude. 255 does the opposite. By default this is 128.

Symmetry along: This selector allows you to pick what type of symmetry you want from JSplacement: Velvet. By default this is set to X & Y.

Use splitter: This selector allows you to turn generation of + shaped overlay on and off. By default this is set to Yes.

Use splitter blending: This selector allows you to turn + shaped overlay's difference blending mode on and off. By default this is set to Yes.

JSplacement: dotGrid - Generator settings

Scale: This slider defines the scale of dots on the map. By default this is set to 30.

Minimum brightness: This slider defines the minimum brightness of the dots, which affects the visibility when used as a mask. By default this is set to 0.

Drawning chance %: Unlike other generators, dotGrid uses systematic co-ordinate based system rather than amount of iterations.
Drawing chance defines the odds of a dot or pattern being drawn per point on co-ordinates.
By default this is set to 2.5.

Pattern max length: The maximum lenght of the patterns. By default this is 5.

Dot shapes: Series of switches for various types of dots. Turning all dots off will be overwritten by box only.
By default all are turned on.

Dot patterns: Series of switches for various types of patterns which use selected dot types. Turning all pattern types off will disable patterns entirely.
By default all are turned on.

Break the code: This switch disables necessary path closing in the code and turns on every single switch. It simulates the accidental broken code during the development of JSplacement 1.1.0.
By default this is turned off.

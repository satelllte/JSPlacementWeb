Custom colorizer gradients

Adding custom gradients can be done in several ways.

The first way is to create a 256x256 image with a left to right gradient of your choice. This can be done in various applications such as Adobe Photoshop and Illustrator, Affinity Photo and Designer, GIMP, Inkscape, etc.

The second way is sampling. Crop and downscale an image to 256x256 size, select a 256x1 area, and scale it up to 256x256 using nearest neighbour interpolation for best result.

The third way is sampling for lazy people. Just use the picture as is. JSplacement's colorizer will sample it for you.

Adding gradients to JSplacement works the same way as custom sprites.
Go to your home directory and open "JSplacement" folder or use "C" hotkey in JSplacement.
Place your gradient images into "CustomGradients" folder.


Supported file formats: JPG, PNG, BMP

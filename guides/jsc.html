JSplacement: Classic - Generator settings

Iterations: This is the main value of JSplacement: Classic. It controls the exact amount of objects drawn onto the canvas.
By default this is set to 1000.

Background brightness: Control background color of the map for different results in whatever you're going to do with it.
In displacement, setting this to 0 will make everything extrude. 255 does the opposite. By default this is 128.

Object types: Series of switches for various types of objects. Turning all objects off will be overwritten by box only.
By default all are turned on.


Object settings

Brightness clamp: Limit generated shape's brightness between two values OR set both sliders to the same value to make it static.
By default these are set to 0 and 255.

Scale: Control shapes' average size. By default it's 100%.

Alpha: Control the opacity of transparent boxes the same way the brightness clamp works.
By default these are set to 50% and 80%.

Spacing: Control the size of the space between the shapes in the patterns.
By default these are set to 100%.

Max Amount: Contol the maximum amount of elements in the patterns.
By default these are set to 5.

Line Width: Control the pixel width of the generated lines.
By default these are set to 10.


Other

Randomize: This button will randomize every single setting in this generator. Have fun!

Presets: This button will open a list of "Windmill's Seal of Approval" presets.

JSplacement: Wyre - Generator settings

Layer count: Defines the amount of overlapping layers of wires from 2 to 255. By default this is 35.

Wires per layer: Number of separate wires drawn per each layer. By default this is 3.
Warning! Setting this values high may cause serious lag on lower end computers. Tread carefully.

Max wire width: Maximum pixel width of the wire. By default this is 64.

Max origin spread: This setting adds maximum size of the random offset to wire's point of origin. By default this is 0.

Min and max step length: Minimum and maximum required distance between two points on the wire. By default these are 128 and 2048.
Warning! Setting these values low may cause serious lag on lower end computers. Tread carefully.

Wire color mode: This controls the way wires' brightness is set. Linear and curve draw them from dark to bright layer by layer.
Inverted linear and inverted curve draw them from bright to dark. Random selects random brightness from 1 to 255.
By default this is set to linear.

Wire width mode: This controls the way wires' pixel thickness is set. Linear and curve draw them from thick to thin layer by layer.
Inverted linear and inverted curve draw them from thin to thick. Random selects random width from 1 to maximum set width.
By default this is set to random.


Step directions

Any: In this mode, wires have 8 different travel directions per step. This is the default value.

Cardinal: In this mode, wires have 4 cardinal directions to travel per each step; up, down, left, and right.

Diagonal: In this mode, wires can only travel diagonally. Essentially same as Cardinal mode, but everything is at 45° angle.

Conical: In this special mode, each wire starts by going one step in any of the 8 direction, and each step afterwards is the same OR +-45° to previous step.
